# Projeto - Biblioteca de Jogos

Criar uma plataforma que permita ao jogador ter controle sobre os seus jogos e consiga organizar sua jogatina.

A jogatina será organizada de acordo com os desafios propostos pelo próprio jogador, isso coloca o jogador no controle o que incentiva a criatividade.

#### Principais funcionalidades

- Sistema de autenticação utilizando Supabase
- Catálogo de jogos da plataforma
- Gerenciamento dos desafios dos jogos
	- Sistema complexo de gerenciamento de estado e de persistência
- Estilização baseada em temas que permite flexibilidade e consistência visual
