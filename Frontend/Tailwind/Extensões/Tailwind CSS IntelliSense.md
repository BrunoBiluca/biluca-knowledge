### Tailwind CSS IntelliSense

[Link da Extensão](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

Adiciona intellisense ao [[VSCode]] do [[Tailwind]] , isso permite utilizar o menu de contexto para exibir classes.

Também autocompleta classes declaradas no tema.
