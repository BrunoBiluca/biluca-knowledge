# Supabase

Supabase é um serviço de [[Backend]] online que disponibiliza várias funcionalidades prontas ao uso como:

- Autenticação
- Banco de dados como serviço (API HTTP)

# Autenticação

O supabase já disponibiliza vários fluxos de utilização do serviço de autenticação como [[React]], [[Next.js]] e React Native.

- [Documentação](https://supabase.com/docs/guides/auth)

```shell
# SDK do supabase
npm install @supabase/supabase-js
```
